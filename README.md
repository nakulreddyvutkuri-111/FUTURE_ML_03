# FUTURE_ML_03 - Resume Screening System

## Project Overview

This project focuses on automatically screening resumes, ranking candidates, and identifying missing skills using NLP and Machine Learning techniques.

---

## Objective

To automate candidate screening and improve recruitment efficiency using resume analysis and skill matching.

---

## Technologies Used

- Python
- Pandas
- Scikit-learn
- NLP
- Matplotlib
- Kaggle Notebook

---

## Key Features

- Resume Parsing
- Skill Extraction
- Job Description Matching
- Candidate Ranking
- Missing Skill Identification
- Visualization

---

## Workflow

1. Data Collection
2. Resume Parsing
3. Skill Extraction
4. Resume Matching
5. Candidate Ranking
6. Visualization

---

## NLP Techniques Used

- TF-IDF Vectorization
- Cosine Similarity

---

## Visualizations

- Candidate Ranking Graph
- Resume Score Comparison

---

## Business Insights

- Reduces manual recruitment effort
- Improves candidate shortlisting
- Helps identify missing skills
- Improves hiring efficiency

---

## Conclusion

The project successfully ranks candidates and identifies missing skills using NLP and machine learning techniques.
